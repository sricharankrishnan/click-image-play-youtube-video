Click Image Play Youtube Video - V.3.0

A Jquery/Javascript plugin thats simple to integrate. Use this to play youtube videos in your web app.

Get To Know More

About This Plugin

The goal is simple. Integrate some files into your webapp and use simple configuration methods to trigger YouTube Video frames playing its video.

Version 2, just like its predecessor, is Open Source and is free to be distributed and used in your projects and app. No strings attached.

Getting Started

You'll need a couple of files to get started with this. Let's get cracking. Start off by downloading all the required assets from this Bundle Link from the Github Project Repo.

First, the Stylesheets. These stylesheets are required and help to provide the basic styling and animation flow for the plugin. The stylesheet is available as Sass Files and Regular CSS files.

Choose to import the Sass file like below (just like any other sass file). The name of the Sass file is youtube-overlay.scss:

@import "./path/to/the/sass/file/youtube-overlay.scss";

Or you can even do this with the traditional link tag like but with say the CSS File:


<link href="/path/to/the/youtube-overlay.css" rel="stylesheet" type="text/css"/>

Next, the Javascript file, this is named as youtube-overlay.js. Keep this in your respective directory, lets say you include it in your web app like so:


<script src="/path/to/the/youtube-overlay.js" type="text/javascript"/>

The Code

Once you've got the files, its time to write the required coded to activate this and play your youtube videos.

Lets imagine for a second that you have a Button element in your web app and you would expect a click event attached to that button to play the Youtube video


<button class="someButtonClass" id="demoButton" data-videourl="some-youtube-url">Demo Button</button>

The button element here as an example, will need to have its id attribute, the data-videourl attribute, which we will use like this along with a callback function. We would need to invoke the required constructor:


var buttonElement = $("#demoButton"),

    idValue = "#" + buttonElement.attr("id"),

    vurl = buttonElement.attr("data-videourl"),

    requiredCallback = function(thisResponse) {

      /* write your code */

    };

Invoke the constructor like so and then use the deploment method as shown:


var demoBuild = new YoutubeOverlayModule({

  sourceUrl: vurl,

  triggerElement: idValue,

  progressCallback: requiredCallback

});

demoBuild.activateDeployment();

You're Done!

If you have been able to understand the above steps in integrating this plugin, you should be fine. Go ahead and click the element like shown in the example by building your own DOM element and then watch the youtube video play.
